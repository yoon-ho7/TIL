### FEAT : 새로운 기능 추가

### FIX : 버그 수정(바르지 않은 동작을 고친 경우)

### REFACTOR : 코드 전면 수정(기능은 변하지 않음)

### CHANGE : 코드 수정

### ADD : 코드, 테스트, 예제, 문서, 이미지 등의 추가

### REMOVE : 코드의 삭제

### STYLE : 코드형식, 정렬, 주석 등의 변경

### DOCS : 문서 수정

### TEST : 테스트 추가

### CHOST : 위에 해당되지 않는 모든 변경

### FILE : 파일의 위치 변경

### DELETE : 파일 삭제
